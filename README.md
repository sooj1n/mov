# mov

### install
```
# main
$ pip install git+https://github.com/sooj1n/mov.git

# branch
$ pip install git_https://github.com/sooj1n/mov.git@<BRANCH_NAME>
```
### start dev
```
$ git clone <URL>
$ cd <DIR>

$ option
$ pdm venv create
$ source .venv/bin/activate
$ pytest
```

### setting env
```bash
cat ~/.zshrc | tail -n 3
# MY_ENV
export MOVIE_API_KEY="<KEY>"
```
